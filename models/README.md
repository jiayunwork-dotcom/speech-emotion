# 预训练模型目录

此目录用于存放预训练的机器学习模型。

## 情感识别模型

期望文件: `emotion_rf_model.pkl`

这是一个使用 pickle 序列化的随机森林分类器模型。

### 模型要求

- 类型: scikit-learn RandomForestClassifier
- 输入特征: 35维声学特征
  - F0均值、标准差、范围 (3维)
  - 语速 (1维)
  - 能量均值、范围 (2维)
  - 频谱质心均值、频谱滚降频率均值 (2维)
  - MFCC前13维的均值和标准差 (26维)
  - 总计: 3+1+2+2+26 = 34维？实际是35维（请检查特征提取代码）

- 输出类别: 6种情感
  - neutral (中性)
  - happy (开心)
  - angry (愤怒)
  - sad (悲伤)
  - fear (恐惧)
  - surprise (惊讶)

### 占位说明

如果 `emotion_rf_model.pkl` 文件不存在，系统将自动使用内置的规则-based 兜底机制进行情感识别。

规则如下:
- 基频标准差大且能量高 → 愤怒
- 基频均值高且变化大 → 开心
- 基频低且能量低 → 悲伤
- 基频范围大且语速快 → 恐惧
- 基频标准差大且均值高 → 惊讶
- 其他情况 → 中性
